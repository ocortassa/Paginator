# Paginator
Page Sequence Generator

This tool can generate a sequence of numbers that represents the sequence of pages to print a folded booklet.

## Usage

> Paginator [document pages number]

## Settings 

### Margin Setting

Top/Down/Right - default, Left - 1cm
